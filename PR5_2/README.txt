Корткий опис проєкту PR5_2:
-------------------------------------------------------------------------------------------------------
Цей проєкт демонструє використання класу CompletableFuture в Java для роботи з асинхронними завданнями.
-------------------------------------------------------------------------------------------------------
Приклад імітує систему бронювання авіаквитків з такими етапами:
 1.Перевірка наявності місць:
  - Асинхронно перевіряє, чи є доступні місця на вибраному рейсі.
 2.Пошук найкращої ціни:
  - Асинхронно шукає найкращу ціну на квиток.
 3.Процес оплати:
  - Завершує бронювання, обробляючи оплату, залежно від результатів перевірки місць та пошуку ціни.
 4.Комбінування та координація завдань:
  - Використовуються методи, такі як thenCombine, thenCompose та allOf, для ефективного управління та
   координації декількох асинхронних завдань.
-------------------------------------------------------------------------------------------------------
Основний функціонал:
 - Демонструє сучасні техніки асинхронного програмування в Java.
 - Імітує реальні сценарії, такі як перевірка місць, пошук цін та обробка оплати.
 - Використовує CompletableFuture для ланцюжків та комбінування асинхронних завдань.
